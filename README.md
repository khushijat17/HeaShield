# 🌡️ HeatShield

A real-time heatwave and water crisis alert platform built for Indore 
and the Malwa region — because the right information at the right time 
can save a life.

## Features
- 🗺️ **Live Heat Risk Map** — zone-wise temperatures across Indore with Danger / Caution / Safe indicators
- 💧 **Water Point Finder** — nearest tankers, hand pumps & govt booths, filterable by type
- ❤️ **Personal Risk Checker** — age & health-based heat guidance
- 📢 **Community Reports** — residents flag water shortages in real time

## Tech Stack
- HTML / CSS / JavaScript (no framework)
- OpenWeatherMap API (live temperature + AQI)
